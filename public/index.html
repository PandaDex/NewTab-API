GET /suggest

Example usage

GET https://new-tab-api.vercel.app/suggest?query=Hello&provider=duckduckgo

Example Response

{ "suggestions": [ {"phrase": "hello world"}, {"phrase": "hello in spanish"}... ] }

Query parameters

query

Type: string

Required: always

Description: The query to search suggestions for.

provider

Type: string

Required: always

Description: The provider to use.

Options: duckduckgo, brave

apikey

Type: string

Required: when using brave as provider

Description: Brave search suggestions API key.
